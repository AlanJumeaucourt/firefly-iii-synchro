Welcome to the Firefly III Synchronization Tool's documentation!
================================================================

This documentation covers the modules and functionalities of the Firefly III Synchronization Tool, 
a Python-based application designed for managing financial data synchronization between Kresus and Firefly III.

Modules
-------

The project is organized into several key modules:

- :doc:`main`: Serves as the entry point of the application, orchestrating the overall application workflow.
- :doc:`utils`: Includes utility functions for various operations within the application.
- :doc:`firefly_api`: Acts as a wrapper for the Firefly III API, facilitating interactions with the API.
- :doc:`kresus`: Responsible for handling and processing Kresus data.
- :doc:`models`: Defines data models representing accounts and transactions.
- :doc:`logging_config`: Contains the configuration for logging across the application.
